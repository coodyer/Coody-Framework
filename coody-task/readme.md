
### Coody Task

### 注意事项：
   Coody-Task 是一款基于Coody Framework而实现的轻量化定时任务插件，遵循cron执行规范。
### 注解：
  @CronTask
  凡是被该注解标记的Bean方法，为其指定cron表达式执行周期，该方法将以定时任务方式围绕指定的cron规则执行
### 功能特色：
    1) 轻量化定时任务体系
### 如何使用
    1、在一个bean方法上添加CronTask注解即可
